◆ $MaxFileSizeToProcess

$MaxFileSizeToProcess = 10

Max size in MB for log and statistics files to process.

This limit is used for files downloaded from the WUI too. The WUI slows down if we try to process larger files. This size should be determined based on the specs of the hardware. The default file size for log rotation is 10 MB for most log files in newsyslog.conf.
