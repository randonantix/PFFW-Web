Log configuration.

This array provides log configuration parameters needed for each module. Detailed behaviour of each module is defined thru the settings in this array.

Parameters
NameTranslatable name of the module.
FieldsArray of columns to show on Logs pages. Parsers may produce more fields than those listed here, e.g. for statistics functions.
HighlightLogsUsed for coloring log lines, holds params for coloring function
HighlightLogs>ColColumn/field to search for keywords (to color the line)
HighlightLogs>TagTag to use in the HTML style, usually 'class', but also 'id'
HighlightLogs>KeywordsContains keywords for red, yellow, and green. Precedence being in that order.
