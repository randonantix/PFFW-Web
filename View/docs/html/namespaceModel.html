 IsInlineAnchor ($str, $force=FALSE)
 Checks and validates any inline rules.
 

Detailed Description

Model of MVC: Generally, the Model is responsible for executing commands from the View.

In PFRE, the Model:

  • Loads rules from strings given to it or from files in the file system
  • Generates rules or rulesets given by the View
  • Validates the values in rules: Untaints all externally supplied data
  • Tests rules using pfctl, and returns the test results
  • Installs, saves, or deletes rule files
  • Authenticates users
  • Configures PFRE settings
